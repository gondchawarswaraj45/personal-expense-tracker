# Xpense – Modern Personal Expense Tracker

Xpense is a premium, glassmorphic web application built to help you track your spending with style and precision. It features a modern dashboard, data visualization, and advanced management tools, all running entirely in your browser.

---

## ✨ Key Enhancements

- **💎 Premium UI**: A sleek "Glassmorphism" aesthetic with smooth transitions and backdrop blurs.
- **🌙 True Dark Mode**: Intelligent theme switching that persists across sessions.
- **📊 Interactive Analytics**: Real-time spending breakdown using **Chart.js**.
- **🏠 Smart Dashboard**: Instant overview of your balance, total income, and total expenses.
- **🔍 Advanced Search & Filter**: Find any transaction by description or filter by categories and date ranges.
- **📥 Export Data**: Download your entire expense history as a **CSV** file for external analysis.
- **📱 Ultra Responsive**: Optimized for every device, from mobile phones to high-res monitors.

---

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript (ES6+), CSS3 (Custom Variables & Backdrop Filters)
- **Framework**: Bootstrap 5 (Base components)
- **Visuals**: Chart.js (Data Visualization), Lucide Icons (Premium iconography)
- **Fonts**: Google Fonts (Outfit)
- **Persistence**: Browser localStorage (JSON-based)

## 🚀 Getting Started

1.  Open `index.html` in any modern web browser.
2.  Start adding your expenses via the **Quick Add** form.
3.  Visualize your spending patterns instantly on the **Dashboard**.
4.  Use the **Expenses** tab to search, filter, and export your transaction history.
